## Постановка задачи

* Выбрать Unix-подобную систему;
* Установить её с помощью одного из возможных способов;
* Установить (или просто запустить пдедустановленные))) несколько программ:
  * Браузер;
  * Текстовый редактор;
  * Графический редактор.
